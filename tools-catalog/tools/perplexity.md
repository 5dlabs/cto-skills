# Perplexity MCP Tools

## Overview
AI search and reasoning engine for complex queries, deep research, and factual answers.

## Capabilities
- AI-powered search with citations
- Deep research with reasoning chains
- Factual question answering
- Multi-step research workflows

## Tool Prefix
`modelcontextprotocol_*` — 4 tools available

## Key Tools
| Tool | Description |
|------|-------------|
| `modelcontextprotocol_search` | AI search with cited sources |
| `modelcontextprotocol_reason` | Multi-step reasoning on a topic |
| `modelcontextprotocol_research` | Deep research with synthesis |
| `modelcontextprotocol_ask` | Direct factual Q&A |

## Configuration
Transport: In-cluster MCP server. Auth: Perplexity API key configured server-side.

## When to Use
- Complex research requiring synthesis of multiple sources
- Factual questions needing cited answers
- Multi-step reasoning about technical topics
- When Tavily results need deeper analysis
