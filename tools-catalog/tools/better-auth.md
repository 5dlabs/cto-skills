# Better Auth MCP Tools

## Overview
Documentation search for Better Auth, the comprehensive TypeScript authentication framework.

## Capabilities
- Authentication pattern documentation
- Integration guides (Next.js, Expo, Electron)
- API reference search

## Tool Prefix
`better_auth_*` — 4 tools available

## Key Tools
| Tool | Description |
|------|-------------|
| `better_auth_search_docs` | Search Better Auth documentation |
| `better_auth_get_guide` | Get integration guide for a framework |
| `better_auth_get_api` | Get API reference for a feature |
| `better_auth_list_plugins` | List available auth plugins |

## Configuration
Transport: In-cluster MCP server.

## When to Use
- Implementing authentication in TypeScript apps
- Integrating auth with Next.js, Expo, or Electron
- Looking up auth API patterns and plugins
