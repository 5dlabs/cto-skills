# GraphQL MCP Tools

## Overview
GraphQL schema introspection and query execution.

## Capabilities
- Schema introspection and type exploration
- Query execution against GraphQL endpoints

## Tool Prefix
`graphql_*` — 2 tools available

## Key Tools
| Tool | Description |
|------|-------------|
| `graphql_introspect` | Introspect a GraphQL schema |
| `graphql_query` | Execute a GraphQL query |

## Configuration
Transport: In-cluster MCP server.

## When to Use

- Testing GraphQL queries
- Understanding available types and fields
