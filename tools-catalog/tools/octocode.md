# OctoCode MCP Tools

## Overview
Advanced GitHub code search, repository structure analysis, and PR archaeology.

## Capabilities
- Semantic code search across repositories
- Repository structure and dependency analysis
- PR history and blame archaeology
- Cross-repo code pattern detection

## Tool Prefix
`octocode_*` — 6 tools available

## Key Tools
| Tool | Description |
|------|-------------|
| `octocode_search_code` | Semantic code search across repos |
| `octocode_get_repo_structure` | Get repository file tree and structure |
| `octocode_search_prs` | Search PR history for code changes |
| `octocode_get_blame` | Get git blame for a file |
| `octocode_find_references` | Find cross-repo references to a symbol |
| `octocode_get_dependencies` | Analyze repository dependencies |

## Configuration
Transport: In-cluster MCP server.

## When to Use
- Finding code patterns across multiple repositories
- Understanding how code evolved via PR archaeology
- Analyzing repository structure and dependencies
- Cross-repo impact analysis before making changes
