# Exa MCP Tools

## Overview
Neural web search optimized for finding code, documentation, and technical content.

## Capabilities
- Neural/semantic web search
- Code-aware search results
- Content similarity search

## Tool Prefix
`exa_*` — 2 tools available

## Key Tools
| Tool | Description |
|------|-------------|
| `exa_search` | Neural web search for code and docs |
| `exa_get_contents` | Get full content of search results |

## Configuration
Transport: In-cluster MCP server. Auth: Exa API key configured server-side.

## When to Use
- Finding code examples and patterns
- Searching for technical documentation
- Finding similar implementations
