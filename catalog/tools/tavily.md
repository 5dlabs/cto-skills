# Tavily MCP Tools

## Overview
AI-powered web search, crawling, and content extraction for research and information gathering.

## Capabilities
- Web search with AI-ranked results
- Web page crawling and content extraction
- Research-grade information gathering
- URL content extraction

## Tool Prefix
`tavily_*` — 5 tools available

## Key Tools
| Tool | Description |
|------|-------------|
| `tavily_search` | AI-powered web search |
| `tavily_crawl` | Crawl a website and extract content |
| `tavily_extract` | Extract content from a specific URL |
| `tavily_research` | Deep research on a topic |
| `tavily_search_context` | Search with context-aware ranking |

## Configuration
Transport: In-cluster MCP server. Auth: Tavily API key configured server-side.

## When to Use
- Researching technical topics and solutions
- Finding documentation and examples
- Extracting content from web pages
- Gathering context for decision-making
